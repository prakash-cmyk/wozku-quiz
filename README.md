# Wozku Quiz Application

A robust Laravel-based Quiz engine supporting 5 distinct question types: Binary, MCQ, Multiple Choice (Checkboxes), Number, and Text.

## 🚀 Features
* **Dynamic Scoring**: Automatically evaluates answers, including complex array-based checkbox logic.
* **Relational Database**: Tracks quizzes, questions, options, user attempts, and individual answers.
* **Responsive UI**: Clean interface built with Blade templates.

## 🛠️ Installation & Setup
1. Clone this repository.
2. Run `composer install` to install dependencies.
3. Copy `.env.example` to `.env` and configure your database settings.
4. Run `php artisan migrate` to build the table structure.
5. Run `php artisan serve` to launch the application.
